Sistema de Informe Diario Automatizado

Descripción General

Sistema automatizado para la generación de informes diarios que integra información del Diario Oficial de Chile, la Comisión para el Mercado Financiero (CMF) y el Servicio de Impuestos Internos (SII).

Este sistema realiza web scraping inteligente, evalúa la relevancia de las publicaciones utilizando IA, y genera informes HTML formateados que se envían automáticamente por correo electrónico.

Características Principales

📰 Diario Oficial

Extrae y clasifica automáticamente las publicaciones del día en:

  • Normas Generales
  • Normas Particulares
  • Avisos Destacados

📊 CMF - Hechos Esenciales

Obtiene automáticamente los hechos esenciales del día con enlaces funcionales garantizados.

  • Enlaces siempre actualizados
  • Formato correcto verificado
  • Información de empresas

🏢 SII - Resoluciones

Recopila las resoluciones tributarias relevantes del Servicio de Impuestos Internos.

  • Filtrado por fecha
  • Evaluación de relevancia
  • Enlaces directos

Tecnologías Utilizadas

Python 3.9+ Django BeautifulSoup4 OpenAI API SMTP Email JSON HTML/CSS

Uso del Sistema

El sistema puede ejecutarse de forma manual o automatizada mediante cron jobs.

Ejecución Manual:

# Generar informe del día actual python generar_informe_completo_simple.py # Generar informe de fecha específica python generar_informe_completo_simple.py "21-07-2025"

Script Principal Recomendado:

# Con integración completa de Django python generar_informe_oficial_integrado_mejorado.py

Ejemplo de Informe

Los informes generados incluyen:

  • Resumen ejecutivo del día
  • Publicaciones relevantes clasificadas por categoría
  • Enlaces directos a documentos originales
  • Información de empresas y montos cuando aplica
  • Formato HTML responsivo y profesional
